## PROJECT OVERVIEW

LocalConvert is a privacy-first file converter forked from VERT.sh (AGPL-3.0). All conversions run locally in the browser via WASM — files never touch a server. Live at https://localconvert.app.

**Tech Stack:**
- Frontend: SvelteKit + TypeScript + Tailwind CSS 3.4
- Hosting: Cloudflare Pages (auto-deploy on git push)
- Image engine: @imagemagick/magick-wasm
- Audio/Video (CPU): @ffmpeg/core@0.12.10 (WASM, CDN-loaded)
- Video (GPU): Mediabunny + web-demuxer (WebCodecs API)
- Documents: Pandoc WASM (hosted on R2, 50MB)
- PDF tools: pdf-lib (MIT) + pdfjs-dist 5.x (Apache 2.0)
- WASM hosting: Cloudflare R2 bucket `localconvert-assets`
- State: Svelte 5 runes ($state, $derived, $derived.by, $effect, $props)
- i18n: Paraglide-JS — 16 locales (ar, ba, de, el, en, es, fr, hr, id, it, ja, ko, pt-BR, tr, zh-Hans, zh-Hant)
- PWA: manifest.json + sw.js + beforeinstallprompt capture
- CSS: SCSS with CSS variables for theming (light/dark mode)
- Styling: Tailwind + custom CSS variables in app.scss + `dynadark:` variant for dark mode
- Fonts: Azeret Mono (display/headings), Lexend (body) via @fontsource
- Icons: Lucide-Svelte

**Repository:** github.com/Sp3llC0d3/localconvert
**Domain:** localconvert.app (Cloudflare)
**Deploy:** `git push origin main` → Cloudflare auto-builds
**Build command:** `npm install --legacy-peer-deps && npm run build`
**Local dev:** `npm run dev` → http://localhost:5174/

## IMPORTANT NOTES

1. **AGPL License** — Must keep VERT attribution in footer. Source link required.
2. **Legacy peer deps** — Always use `npm install --legacy-peer-deps` or builds fail.
3. **COOP/COEP headers removed** — Broke CDN WASM loading. SharedArrayBuffer disabled.
4. **Pandoc WASM on R2** — 50MB file at `https://pub-a63702e42341425494caded68a9378bd.r2.dev`
5. **web-demuxer needs absolute URL** — `${window.location.origin}/web-demuxer.wasm`
6. **Svelte 5 runes** — Project uses $state, $derived, $derived.by, $effect, $props (not legacy stores)
7. **Deploy** — Just `git push origin main` — Cloudflare auto-builds and deploys.
8. **Dark mode** — Uses `dynadark:` Tailwind variant, respects system preference + manual toggle in localStorage.
9. **UI language** — Use "select" not "upload", "save" not "download" in all user-facing copy.
10. **Trust badge** — "No server uploads" pill must stay exactly as-is. Never change it.
11. **i18n workflow** — After adding keys to `messages/en.json`, run `npx @inlang/paraglide-js compile` to regenerate compiled messages. All 16 locales must stay in sync.
12. **Node.js scripts** — Must use `.cjs` extension (project has `"type": "module"` in package.json).
