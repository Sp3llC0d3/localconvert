## PROJECT OVERVIEW

LocalConvert is a privacy-first toolkit for files and code, forked from VERT.sh (AGPL-3.0). 36 tools + 200+ format conversions — all running locally in the browser via WASM. Files never touch a server. Live at https://localconvert.app.

**Tech Stack:**
- Frontend: SvelteKit + TypeScript + Tailwind CSS 3.4
- Hosting: Cloudflare Pages (auto-deploy on git push)
- Image engine: @imagemagick/magick-wasm
- Audio/Video (CPU): @ffmpeg/core@0.12.10 (WASM, CDN-loaded)
- Video (GPU): Mediabunny + web-demuxer (WebCodecs API)
- Documents: Pandoc WASM (hosted on R2, 50MB)
- PDF tools: pdf-lib (MIT) + pdfjs-dist 5.x (Apache 2.0) + pdf-lib-plus-encrypt (MIT, for password protection)
- Presentations: pptxgenjs (MIT, for PDF → PPT)
- WASM hosting: Cloudflare R2 bucket `localconvert-assets`
- State: Svelte 5 runes ($state, $derived, $derived.by, $effect, $props)
- i18n: Paraglide-JS — 28 locales (ar, ba, de, el, en, es, fa, fr, hi, hr, id, it, ja, ko, ms, nl, pl, pt, pt-BR, ro, ru, sv, th, tr, uk, vi, zh-Hans, zh-Hant)
- PWA: manifest.json + sw.js + beforeinstallprompt capture
- CSS: SCSS with CSS variables for theming (light/dark mode)
- Styling: Tailwind + custom CSS variables in app.scss + `dynadark:` variant for dark mode
- Fonts: Azeret Mono (display/headings), Lexend (body) via @fontsource
- Icons: Lucide-Svelte

**Repository:** github.com/Sp3llC0d3/localconvert
**Domain:** localconvert.app (Cloudflare)
**Deploy:** `git push origin main` → Cloudflare auto-builds
**Build command:** `npm install --legacy-peer-deps && npm run build`
**Local dev:** `npm run dev` → http://localhost:5174/
**Sitemap:** 226 URLs — generated by `scripts/generate-sitemap.js`

### Interaction Patterns
- **Live canvas preview** — rAF loop redraws on every setting change, base image cached as ImageData
- **Click-to-place** — Ghost element follows cursor, click places, drag repositions (HTML overlays, not canvas)
- **Undo/redo** — Generic history system (`src/lib/util/history.svelte.ts`), Ctrl+Z/Ctrl+Shift+Z
- **Keyboard shortcuts** — Delete, Escape, Arrow keys (1pt nudge, Shift=10pt) on PDF Edit
- **Draggable handles** — PDF Crop has 4 edge handles synced with number inputs
- **PDF preview utility** — `src/lib/pdf/preview.ts` with `loadPdfDocument()` + `renderDocPageToCanvas()` for single-parse, multi-page preview

## IMPORTANT NOTES

1. **AGPL License** — Must keep VERT attribution in footer. Source link required.
2. **Legacy peer deps** — Always use `npm install --legacy-peer-deps` or builds fail.
3. **Svelte 5 runes** — $state, $derived, $derived.by, $effect, $props (not legacy stores)
4. **Deploy** — Just `git push origin main` — Cloudflare auto-builds and deploys.
5. **Dark mode** — `dynadark:` Tailwind variant, system preference + manual toggle.
6. **UI language** — "select" not "upload", "save" not "download" in all user-facing copy.
7. **Trust badge** — "No server uploads" pill must stay exactly as-is.
8. **i18n workflow** — After adding keys to `messages/en.json`, add to all 27 other locales too, then compile. All 28 locales must have 788 keys each.
9. **Node.js scripts** — Must use `.cjs` extension (project has `"type": "module"`).
10. **PDF preview** — Use `loadPdfDocument()` + `renderDocPageToCanvas()` for efficient multi-page preview. Always call `doc.destroy()` on cleanup.
11. **Live preview pattern** — Cache base image as ImageData, redraw overlay via rAF. No debouncing needed.
12. **Image validation** — ImageUploader validates file type + size (>100MB blocked, >20MB warned).
13. **Scoped CSS** — Svelte scopes styles per-component. Use Tailwind utilities (global) for Panel class props.
